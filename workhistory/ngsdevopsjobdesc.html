Duties and Responsibilities

  • Manages the build processes end-to-end, from the planning of builds through actual releases (deployment to production), including planning the deployment of software releases of products and the in-house applications
  • Maintains the current Nowcom change management process for documenting software build/release
  • Works with the end-users and internal cross-functional teams to prioritize and scope scheduled builds with completion dates for each phase
  • Documents concisely and fully, and appropriately distributes all software builds
  • Maintains the current Microsoft Azure DevOps Service version control systems which include user maintenance (as well as establishing the appropriate folder structure and branches for builds), creates and maintains version directories, and manages source control branches
  • Validates binaries for custom .NET, .NET Core, WordPress, and PHP developments as necessary, and moves report files and non-database code to the production environment
  • Coordinates and participates in regular post-implementation reviews and presents build metrics in order to assist with the evaluation of release quality
  • Configures and implements automated tools and/or manual processes that integrate the OctopusDeploy app
  • Works with other technical team members to resolve all build/release related issues
  • Provides input to the Quality Assurance function when planning the testing process, and assists with the planning and support of user acceptance testing (UAT)
  • Writes and maintains written procedures for build/release management
  • Works flexible hours to avoid impact to production systems
