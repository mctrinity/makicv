Duties and Responsibilities

  • New Projects Department (R&D)

    • Undertakes software research and testing for fast statistics/counts software, deduplication and merge/purge software, data hygiene and report-generating software, and developing data processing software
    • Undertakes testing of other vendors’ data processing software
    • Creates test scripts/scenarios for Data Services and Business Intelligence department end-users
    • Creates simple training manuals/documentation for end-users’ perusal

  • Data Services Department

    • Analyzes and maintains databases of up to 15 million records for UK customers for their direct marketing
    • Tools used:
      • ETL using MS SQL / FoxPro for Windows
      • Data hygiene and file/report creation
      • Deduplication and merge/purge software from UK vendors
    • Database file format conversion
    • Publishes updated databases using web-based software for clients use
